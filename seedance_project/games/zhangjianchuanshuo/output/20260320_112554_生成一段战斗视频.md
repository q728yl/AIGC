## 1. Director's Thought
The user only asked for a battle video, so I anchored the whole shot on the provided first frame to preserve the original vertical poster-like composition, faction arrangement, hero position in the lower-right, UI position at the bottom, and the bottom-to-top diagonal action axis. I selected one core combat event only: the main hero performs a charged upward slash and magic burst, so the motion stays slow-to-fast and readable, which follows the 'slow is smooth' rule and reduces deformation risk. I used dedicated action sheets for the hero and enemy, then separated casting VFX, impact explosion VFX, and UI assets into layers so the battle feedback loop is complete: charge-up, release, hit, recoil, damage numbers, and HP reduction. The style is kept faithful to Sword Legend: bright fantasy gacha, stylized 3D chibi combat, rich particles, clean commercial mobile-game polish, not realistic.
