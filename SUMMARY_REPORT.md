# 🎬 Seedance AI Director - 阶段性成果总结报告

截至目前，我们已经成功搭建并完善了 **Seedance AI Director**（智能游戏视频生成与素材创作Agent）的核心工作流。通过将视觉模型、文本模型、自动化爬虫和视频生成API深度整合，实现了从“单张截图+一句话指令”到“完整高质量视频”的端到端自动化。

以下是目前实现的核心路径与关键功能总结：

## 🌟 核心工作流 (Pipelines)

系统现在支持三种维度的生成路径，以应对不同的创作需求：

### 1. Plan A：纯图生图与逻辑推演路径 (Image-based Pipeline)
这是系统的基础能力，Agent 通过分析传入的**单张游戏首图（参考图）**，结合用户的文本指令，完成以下工作：
- **画面感知与逻辑锁定 (Perception Lock)**：深度理解首图的画风（像素、3D、二次元等）、视角、角色状态和UI布局。
- **资产逆向工程 (Asset Reverse Engineering)**：判断当前画面缺少哪些素材才能完成用户的指令（如：缺少攻击特效、受击UI、飘字等）。
- **自动化素材绘制**：调用 `GPT-Image-1.5`，通过严格的 Prompt 约束（确保不改变原图画风和核心角色设计），自动绘制出缺失的 Sprite Sheet（序列帧）和 UI 元素切图。
- **Timeline 剧本编排**：将首图和新生成的素材进行层级组合，生成严谨的 `[00s-xxs]` 时间轴剧本，确保逻辑与首图完美承接。

### 2. Plan B：自动化视频参考路径 (Video-reference Pipeline)
为解决“AI生成视频容易出现动作不自然或缺乏游戏特定表现力”的问题，引入了这套基于真实视频的爬虫检索系统：
- **智能意图拆解与关键词生成**：大模型根据用户的指令，自动生成用于搜索视频的关键词集合。
- **全自动 TapTap 爬虫与下载**：使用 `Playwright` 和 `yt-dlp`，自动搜索并下载前几个相关的真实玩家演示视频。
- **多级回退机制**：如果主关键词搜不到，自动使用备用关键词重试，确保极高的视频获取成功率。
- **大模型关键帧评比选优**：提取多个候选视频的关键帧交由 MLLM 评估。通过**严苛的过滤规则**（拒绝静止画面、纯文本菜单等），强制选出动态张力最好、最契合用户意图的一个视频，并精准裁剪出最精彩的几秒片段（通过 `FFmpeg`）。

### 3. Plan C：全混合自动化工作流 (Full Hybrid Pipeline) **[推荐模式]**
这是目前最强大的模式，它完美融合了 Plan A 和 Plan B 的优势，且互不干扰：
- **互不污染原则**：找来的参考视频**绝不会**影响图像素材的生成风格，图像素材依然 100% 忠实于你提供的首图，保证画风不跑偏。
- **神形分离原则**：在最终编写 Timeline 剧本时，大模型被**强制约束**：只提取参考视频中的“动作技巧”（如卡肉感、镜头震动、UI弹出节奏、战斗韵律等），而**绝对不照搬**视频里的具体故事线或特定角色动作语义。一切故事情节依然由你的“原始指令”驱动。
- **本地视频直通车**：支持跳过爬虫，直接在命令行 (`--video`) 或网页端传入本地准备好的完美参考视频。

## 🛠 系统与工程整合亮点

### 1. 无缝对接 Seedance 2.0 API
- 实现了与火山引擎/Seedance 2.0 API 的深度集成。
- **动态参数提取**：系统会自动解析生成的 Timeline 剧本，精确计算出所需的视频长度（限制在 4-15s）；会自动读取首图尺寸，算出最贴合的 `ratio`（如 16:9, 9:16），真正做到智能化提参。
- **突破内网限制**：由于 API 不支持本地视频 Base64 传输，开发了临时图床上传机制（通过 `tmpfiles.org`），将裁剪好的本地参考视频秒传公网并获取 URL 交给 API 处理。
- **自动提交与轮询**：流程结束后自动提交生流任务，并可随时通过 `check_task.py` 脚本轮询下载链接，防止因等待过久导致网页超时。

### 2. 可视化 Web 工作台 (Web UI)
我们开发了一个轻量但功能齐全的 Flask 前端交互页面 (`http://127.0.0.1:5050`)：
- **多模态输入**：支持同时拖拽上传“参考首图”和“参考视频”，并输入自然语言指令。
- **实时日志反馈**：在页面上可以实时看到 Agent 正在思考什么、正在画哪张图、正在搜哪个关键词。
- **排版优化的结果展示**：
  - 生成结束后，顶层会并排展示**参考视频**和**任务提交状态（或生成的最终视频）**。
  - 下层画廊会整齐地排列出所有基于首图新生成的序列帧和特效切图。
  - 核心提示词（中英文对照版本）会以清晰的 Markdown 格式渲染展示。

## 🎯 总结
目前的 Agent 已经具备了极高的自主性和可用性。它不仅是一个“会写提示词”的脚本，而是一个真正的**导演**：它能看懂你的图，知道缺什么素材并自己去画；它能为了动作张力去网上找参考视频并自己剪辑；最重要的是，它“听得懂话”，能严格区分什么是“风格约束”、什么是“动作参考”，最终自动帮你把所有参数和素材打包，一键丢给视频大模型生成最终成片。
